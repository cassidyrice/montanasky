# MontanaSky SOP Knowledge Base

This folder contains markdown SOPs used by the MontanaSky Support Copilot.

The files are written for retrieval-augmented generation and should stay structured, direct, and consistent.

## File pattern

Each SOP should include:

- When to use this SOP
- Tools needed
- Step-by-step workflow
- Safety checks
- Escalation triggers
- Ticket note template
