# SOP: CPE Inventory / Recycle / Return Process

## When to Use
Use when equipment has been returned, replaced, removed from a customer account, or needs to be marked available/recycled.

## Tools Needed
- Admin
- CPE Inventory
- UISP if radio/router assignment exists there
- Ticketing system

## Step-by-Step Process
1. Confirm customer/account/location in Admin.
2. Identify the equipment involved.
3. Match equipment by device type, MAC, serial number, name, or inventory label.
4. Confirm whether the equipment is active, inactive, returned, damaged, or replaced.
5. If no damage is present, recycle or mark the correct equipment available in the appropriate inventory system.
6. If equipment exists in UISP, confirm whether it should stay assigned or be removed.
7. Do not remove equipment from a customer if it may still be active.
8. Document what was returned, removed, recycled, or marked available.

## Safety Checks
- Never recycle equipment without matching device identifiers.
- Do not rely only on customer name.
- Check MAC/serial/device type.
- Confirm replacement equipment is active before marking old equipment available.
- If equipment is damaged, follow internal damaged-equipment handling rather than marking available.

## Ticket Note Template
```text
Processed CPE inventory update for [Customer name]. Confirmed correct account/location in Admin. Matched equipment by [MAC / serial / device type / name]. Equipment status: [returned / replaced / inactive / damaged]. Inventory action completed: [marked available / recycled / left assigned / escalated]. Notes: [details].
```
